## Overview

To begin with Nyao, create a new documentation project from your dashboard.

## Steps

1. Go to your Nyao dashboard
2. Click "New Project"
3. Enter the project name and description
4. Select a template (optional)
5. Click "Create"

Your project workspace will appear, ready to organize docs.
