## Upgrading Nyao

Follow these steps to upgrade to the latest version.

<Steps>
  <Step title="Backup Your Data" icon="database">

  Export all spaces:

  ```
  nyao export --all-spaces backup.json
  ```

  </Step>
  <Step title="Update Package" icon="download">
    <Tabs>
      <Tab title="npm" icon="package">
````bash
npm install nyao@latest
````
      </Tab>
      <Tab title="Docker" icon="docker">
````bash
docker pull nyao:latest
docker run -p 3000:3000 nyao:latest
````
      </Tab>
    </Tabs>
  </Step>
  <Step title="Run Migrations" icon="settings">
    ```
    nyao migrate
    ```
  </Step>
  <Step title="Restart Server" icon="refresh-cw">
    Restart your Nyao instance and verify at `http://localhost:3000`.
  </Step>
</Steps>

<Callout kind="tip">
After upgrading, clear your browser cache to ensure all new features load correctly.
</Callout>
